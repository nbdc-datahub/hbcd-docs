# How To Cite HBCD Data Release Docs

## Citation Instructions

The HBCD Data Release Documentation is version-controlled, allowing researchers to cite and view specific snapshots of the website attached to each release. This includes both major and minor data release (e.g. Release 1.0 vs Release 1.1). Because the documentation is updated on both an ongoing basis and well as with new releases, citing the version will guide others to the specific material that was consulted. To cite the current version, visit the most recent release on [Zenodo](https://doi.org/10.5281/zenodo.15002829). 

The appropriate citation is listed in the **Citation** box on the right side of the page—simply copy and paste it into your references.
